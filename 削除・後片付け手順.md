# 削除・後片付け手順

このアプリをやめる／消すときの手順です。
**全部消す必要はありません。** 消したいものだけ、その章だけやればOKです。
（上から順に「影響が小さい→大きい」順に並べています）

---

## 0. まず知っておくこと

このアプリは、次の5か所に「物」が存在します。消すときはそれぞれ別の作業になります。

| 場所 | 何があるか | 消すと何が起きる |
|------|-----------|------------------|
| ① ブラウザ | 学習履歴（正答率など） | 履歴だけ消える。アプリは残る |
| ② このPCのフォルダ | アプリのソースコード | PC上の編集環境が消える |
| ③ Vercel | 公開中のURL | 外出先からアクセス不可になる |
| ④ GitHub | アップロードしたコード | クラウド上のバックアップが消える |
| ⑤ Node.js | アプリを動かすソフト | 他のアプリ開発もできなくなる（普通は残してOK） |

---

## 1. 学習履歴だけリセットしたい（アプリは残す）

一番軽い操作。「最初からやり直したい」ときはこれだけでOK。

- **方法A:** アプリのトップ画面 右上の **「履歴リセット」** ボタンを押す
- **方法B:** 使っているブラウザで履歴を消す
  - PC: F12 → 「Application」→ 左の「Local Storage」→ 該当サイトを右クリック →「Clear」
  - スマホ: ブラウザの設定 →「サイトデータを削除」

> PCとスマホ、Vercel版とローカル版は、それぞれ履歴が独立しています（別々に消す必要あり）。

---

## 2. ローカルのアプリを止める／PCから消す

### 2-1. 起動を止めるだけ
- `npm run dev` を実行中のターミナルで **Ctrl + C** を押す → サーバー停止
- （PCを再起動・シャットダウンしても止まります）

### 2-2. PCからフォルダごと完全に消す
1. 念のためアプリを止める（Ctrl + C）
2. エクスプローラーで `java-silver-quiz` フォルダを **そのままゴミ箱へ** ドラッグ
3. これでPC上のアプリは消えます

> ⚠️ フォルダを消しても、③Vercelの公開URLと④GitHubのコードは**残ります**。完全にやめるなら3・4も実施してください。

---

## 3. Vercelの公開を消す（外出先URLを止める）

「もう外からアクセスさせたくない」ときはこれ。

1. **https://vercel.com** にログイン
2. ダッシュボードで **`java-silver-quiz`** プロジェクトをクリック
3. 上のタブ **「Settings」** を開く
4. 一番下までスクロールして **「Delete Project」**（赤いエリア）
5. 確認のためプロジェクト名の入力を求められるので、`java-silver-quiz` と入力 → 削除
6. 発行されていた `https://....vercel.app` のURLは無効になります

> これでVercelの無料枠の使用も完全に止まります（元々無料なので料金は発生しませんが、念のため）。

---

## 4. GitHubのリポジトリを消す

クラウド上のコードのバックアップも消したいとき。

1. **https://github.com/abcryo1008-debug/java-silver-quiz** を開く
2. 上のタブ **「Settings」** を開く
3. 一番下の **「Danger Zone」** までスクロール
4. **「Delete this repository」** をクリック
5. 確認のため `abcryo1008-debug/java-silver-quiz` の入力を求められるので、そのとおり入力 → 削除

> ⚠️ Vercelと連携している場合、先に **3（Vercel削除）→ 4（GitHub削除）** の順がおすすめ。GitHubを先に消すとVercel側がエラー表示になることがあります（消すだけなら順序はどちらでも可）。

---

## 5. Node.js をアンインストールする（基本は不要）

> 💡 **普通は残しておくのがおすすめ。** 他のWebアプリ開発にも使う共通ソフトなので、消すと再セットアップが面倒です。「PCから完全に何もかも消したい」ときだけ実施してください。

1. Windowsの **「設定」→「アプリ」→「インストールされているアプリ」**
2. 検索欄に `Node` と入力
3. **「Node.js」** を見つけて「⋯」→「アンインストール」
4. 画面の指示に従って削除

（git や VSCode も同様に、設定 → アプリ から個別にアンインストールできます。これらも他で使うので普通は残してOK。）

## まとめ：目的別チートシート

| やりたいこと | やる章 |
|------------|--------|
| 成績だけ最初から | 1 |
| 今日はもう閉じる | 2-1（Ctrl+C） |
| PCからアプリを消す | 2-2 |
| 外出先URLを止める | 3 |
| クラウドも含め完全に消す | 2-2 → 3 → 4 |
| PCをまっさらに戻す | 2-2 → 3 → 4 →（必要なら）5 |
